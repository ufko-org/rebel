Rebel Built-In Functions in Groups

Some functions appear in more than one group.

---

List of groups:

#1  List processing, flow control, and integer arithmetic
#2  String and conversion functions
#3  Floating point math and special functions
#4  Matrix functions
#5  Array functions
#6  Bit operators
#7  Predicates
#8  Date and time functions
#9  Statistics, simulation and modeling functions
#10 Pattern matching
#11 Financial math functions
#12 Input/output and file operations
#13 File and directory operations
#14 Processes and the Cilk API
#15 HTTP networking API
#16 Socket TCP/IP, UDP and ICMP network API
#17 Reflection and customization
#18 System functions
#19 Importing libraries
#20 Rebel internals API

---

Parameter naming convention:

seq       list / string / array  
seq-1     first sequence  
seq-2     second sequence  
num       count / length / range  
num-1     count value 1  
num-2     count value 2  
int       integer (mathematical)  
float     floating-point (mathematical)  
str       string  
str-1     first string  
str-2     second string  
str-pat   pattern string  
sym       symbol  
ctx       context  
func      function / lambda  
pred      predicate  
data      raw data  
path      filesystem path  
host      network host  
port      network port  
sock      socket descriptor  
matrix    matrix value

Each function block contanins the following fields:

name      (syntax)
ret       descripton
